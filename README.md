# ClearCost AI Phone Prototype v1.1

IMPORTANT:
Do not test this by opening index.html inside a ZIP preview, ChatGPT file preview, iPhone Files preview, or Quick Look.
Those viewers may display the page but block JavaScript navigation.

Reliable testing:
1. Extract the ZIP.
2. Publish the folder to Cloudflare Pages, GitHub Pages, Netlify, or another static web host.
3. Open the resulting https:// address on the phone.

Computer-only quick test:
- Extract the ZIP.
- Open index.html in Chrome or Edge.
- If local-file restrictions interfere, serve the folder with a local web server.

Changes in v1.1:
- Removed service-worker registration during prototype testing.
- Improved touch behavior.
- Added a warning when opened as a local file.
