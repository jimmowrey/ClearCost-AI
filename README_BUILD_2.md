# ClearCost AI Build 2

Adds the first runnable mobile-first application shell.

## Files
- index.html
- css/app.css
- js/app.js

## Working capabilities
- Home
- New Analysis
- Multiple PDF selection
- Drag-and-drop PDF support
- Statement queue
- File name and size display
- Duplicate-file prevention
- Remove or clear files
- Statement Validation screen
- History
- Settings
- Mobile navigation

Unavailable processing stages display `Waiting for statement processing`. No OCR, processor, page-count, or financial values are fabricated.
